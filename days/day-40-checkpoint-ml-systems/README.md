## üéØ Hands-On Assessment Exercise

### Mini-Project: ML System Health Check

**Objective**: Demonstrate your ability to assess and improve an existing ML system

**Scenario**: You've inherited an ML system for customer churn prediction that has been running in production for 6 months. The system is experiencing performance issues and the business team is questioning its effectiveness.

**Your Task**: Conduct a comprehensive health check and provide recommendations

#### Step 1: System Analysis (15 minutes)
Review the following system components and identify potential issues:

```python
# Simplified system architecture
system_components = {
    "data_pipeline": {
        "last_updated": "2 weeks ago",
        "data_quality_score": 0.75,
        "missing_values": "15%",
        "schema_changes": 3
    },
    "feature_store": {
        "feature_freshness": "6 hours",
        "feature_drift_score": 0.12,
        "feature_coverage": "85%",
        "serving_latency": "150ms"
    },
    "model_performance": {
        "current_auc": 0.78,
        "baseline_auc": 0.85,
        "prediction_volume": "50K/day",
        "error_rate": "2%"
    },
    "monitoring": {
        "alerts_last_week": 15,
        "dashboard_uptime": "92%",
        "log_retention": "7 days",
        "drift_detection": "disabled"
    }
}
```

**Questions to Answer**:
1. What are the top 3 issues you identify?
2. Which issue should be prioritized first and why?
3. What additional information would you need to gather?

#### Step 2: Improvement Plan (15 minutes)
Create a prioritized action plan to address the identified issues:

**Template**:
```markdown
## Issue Priority Matrix

### Critical (Fix Immediately)
- Issue: [Description]
- Impact: [Business/Technical Impact]
- Solution: [Specific Actions]
- Timeline: [Estimated Time]

### High Priority (Fix This Week)
- Issue: [Description]
- Impact: [Business/Technical Impact]
- Solution: [Specific Actions]
- Timeline: [Estimated Time]

### Medium Priority (Fix This Month)
- Issue: [Description]
- Impact: [Business/Technical Impact]
- Solution: [Specific Actions]
- Timeline: [Estimated Time]
```

#### Step 3: Implementation Strategy (10 minutes)
Outline how you would implement the top priority fix:

1. **Technical Approach**: What specific technologies/methods would you use?
2. **Risk Mitigation**: What could go wrong and how would you prevent it?
3. **Success Metrics**: How would you measure the success of your fix?
4. **Rollback Plan**: What would you do if the fix causes issues?

### Assessment Criteria

Your response will be evaluated on:
- **Problem Identification** (30%): Ability to spot critical issues
- **Prioritization** (25%): Understanding of business and technical impact
- **Solution Design** (25%): Practical and implementable solutions
